Card Deck

Card Deck is a simple API built in Python using the Flask framework. It provides suffled and unshuffled card stacks built on a variable number of standard 52 card decks, specified by the user.

Card Deck provides the following routes:

/deck/[count]

The basic /deck route, with the optional count parameter, provides a suffled deck of cards based on the standard 52 card deck, without jokers. You can increase the number of decks used by appending the desired number as the count parameter. For example, /deck/6 will return a shuffled deck built from six 52 card decks, for a total of 312 randomly ordered cards.

/unshuffled/[count]

If you want a set of unshuffled cards, the /unshuffled route provides that, with the optional count parameter that functions as above. The deck is returned in ascending order of value with Ace counted as the low card. The suits are ordered Hearts, Clubs, Spades, and finally Diamonds, with duplicate cards from a multi-deck appearing adjacent to one another.

Jokers

Jokers are available in single deck sets either unshuffled or unshuffled. A shuffled deck with jokers included can be obtained from the /deck/jokers route, and /unshuffled/jokers provides an unshuffled deck with two jokers added at the end of the stack. It is only possible to include jokers in a single deck.
